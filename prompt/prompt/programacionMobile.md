# Prompt mobile Atlas — React Native + Expo

Usa React Native + Expo + TypeScript.

## Reglas críticas

- Tokens en SecureStore.
- No subir contactos completos.
- Pedir permisos con contexto.
- API por servicios, no fetch en pantallas.
- Manejar doble tap y reintentos.
- KYC por adapter.
- Push notifications controladas.

- Consentimientos.
- KYC.
