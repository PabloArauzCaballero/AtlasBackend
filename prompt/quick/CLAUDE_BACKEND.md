# CLAUDE quick — Backend Atlas

Backend obligatorio: NestJS + TypeScript + Sequelize + PostgreSQL + Zod + JWT.

Lee `BACKEND_DEVELOPMENT_CONTEXT.md` antes de escribir código.

No usar Express puro, FastAPI, Prisma, TypeORM ni Alembic.

Controllers delgados, services con casos de uso, repositories con Sequelize, schemas con Zod, mappers para respuestas.

Reglas Atlas: 60% inicial, 40% en 3 cuotas, 14 días, no acelerar deuda, snapshot de riesgo, límite por movimientos, auditoría.

Si falta política crítica, pregunta.


Regla de pendientes: todo pendiente, bloqueo o supuesto debe quedar documentado en Markdown (`PENDIENTES_ATLAS.md` o `docs/pending/pending-items.md`).
