# Carpeta prompt

Esta carpeta contiene la versión modular de los prompts para adjuntar a una IA.

Orden recomendado:

1. `index.md`
2. `programacionGeneral.md`
3. Archivo específico:
   - `programacionBackend.md`
   - `programacionFrontendWeb.md`
   - `programacionMobile.md`
   - `programacionInfra.md`
4. `../PROJECT_BRIEF_ATLAS.md`
5. `../CHECKLIST_FINAL.md` si se pide entrega final.


## Pendientes

`PENDIENTES.md` define la regla obligatoria para registrar todo pendiente, supuesto, bloqueo o decisión abierta en Markdown. Debe leerse junto con `index.md` antes de generar código o documentación.
